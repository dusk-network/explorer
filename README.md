# Explorer
Dusk block explorer

This is the temporary repository for building the Dusk block explorer. The application that is built here will be migrated into the [Rusk repository](https://github.com/dusk-network/rusk).

## NPM Scripts

* `npm run build`: creates the production build.
* `npm run check` runs the type checker
* `npm run check:watch` runs the type checker in watch mode
* `npm run dev`: generates the development build and starts the dev server
* `npm run lint`: runs linting for code and test, in sequence.
* `npm run lint:code`: runs linting for code only.
* `npm run lint:tests`: runs linting for tests only.
* `npm run preview`: previews the production build.
* `npm test` runs the test suite
* `npm run test:watch` runs the test suite in watch mode
