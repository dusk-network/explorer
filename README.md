# Explorer
Dusk block explorer

This is the temporary repository for building the Dusk block explorer. The application that is built here will be migrated into the [Rusk repository](https://github.com/dusk-network/rusk).

```bash
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```

## Building

To create a production version of your app:

```bash
npm run build
```

You can preview the production build with `npm run preview`.
